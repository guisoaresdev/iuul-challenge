# iuul-challenge

Contém o desafio 1 do processo de residência da iUUL

## Installing

```
git clone https://github.com/guisoaresdev/iuul-residence
cd iuul-residence
npm install
```

## Desafio 1 - Node.js/Express

Running questao1.js

```
npm run questao1
```
